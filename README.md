# EV Registration Insights

## 프로젝트 실행 방법

1. **가상 환경 활성화**  
   - **Windows**:
     ```bash
     .\venv\Scripts\activate
     ```
   - **macOS/Linux**:
     ```bash
     source venv/bin/activate
     ```

2. **필요한 패키지 설치**  
   ```bash
   pip install -r requirements.txt

3. 프로젝트 실행 
   ```bash
   python3 app/app.py

4. 접속
   ```bash
   http://localhost:5000
